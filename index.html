Ce plugin Sarah permet d'utiliser le logiciel Sms Gateway sur un telephone Android.


Materiel necessaire :


1 : Un vieux telephone Android, avec ou sans batterie
2 : Installer le logiciel Sms Gateway (gratuit) sur celui-ci
3 : Un cordon Usb, ou un chargeur pour laisser le telephone en charge
4 : Un forfait mobile Low cost (genre free pour ne pas le citer a 0euros)
5 : PENSEZ A INSTALLER LE PLUGIN ASKME de Veyret Jerome : c'est une dependance obligatoire a partir de la version 0.3 !

Prerequis :


Installez le logiciel sms gateway comme indique ici
Pensez a indiquer "http://IPSARAH:8080/sarah/sms" dans la case "Forward incoming sms to http"puis
Penser a cocher la case "forward incoming sms to http"
Donnez une IP FIXE a votre mobile en le connectant a votre reseau wifi
Allumez bien le serveur sms sur le telephone
Parametrez dans le fichier custon.prop l'ip et le port de votre Gatewaysms
Dans le fichier sms.xml, pensez a introduire le prenom et le numero de telephone correctement

ATTENTION :


il sagit d'une version Beta. Tout fonctionne correctement excepte que :
lorsque l'on envoie un message vocalement, le contenu du message n'est pas toujours bien reconnu, parlez de maniere claire et forte
certains tests de debug n'ont pas ete faits, mais en theorie le foncionnement de base ne devrait pas poserde probleme



Comment utiliser le plugin ?



- En interne, un plugin sarah doit faire appel a : http://127.0.0.1:8080/sarah/sms?phone=NUMBER&text=MESSAGE
- Ecrire un message vocalement : Dites Sarah envoi un message a PRENOM disant MESSAGE
Bien sur, il faudra remplir correctement le fichier sms.xml
Sarah va vous relire votre message, et si il correspond a votre souhait vous dites"OUI" pour envoyer ou "NON" pour annuler
- La reception de SMS est simple : Vous recevez un sms, Sarah vous donne le numero du correspondant et vous demande
si vous voulez lire le message, dites "OUI" ou "NON".
